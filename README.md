Site Showcase
============

This plugin implements a website showcase often called a CSS gallery. It is a gallery containing website screenshots where users can share and rate their sites.

See [Live Demo](http://www.cotonti.com/showcase/)

Authors: [Vladimir Sibirov](https://github.com/trustmaster), [Kalnov Alexey](https://github.com/Alex300), Cotonti Team

Plugin page:  
https://www.cotonti.com/extensions/files-media/site-showcase

## Features
- Automatic thumbnails via [Api Flash](https://apiflash.com/)
- Comments and ratings
- Tags
- Admin validation of all submissions

## Install and customize
- Register yourself a [Api Flash](https://apiflash.com/register) account
- Unpack the plugin into your Cotonti plugins folder
- Install it in Administration => Plugins
- Go to plugin configuration and set options there. Don't forget to copy access key from you [Api Flash](https://apiflash.com/dashboard/access_keys) account
- If you want to customize the layout, copy showcase.tpl and showcase.display.tpl from `plugins/showcase/tpl` to `themes/your_theme/plugins` folder and edit them to fit your skin
- It comes with English and Russian langfiles, to add more, go to `plugins/showcase/lang` and translate it into your language
